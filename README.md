# Installation
- Clone the project
  bash
     git clone https://github.com/fahrisyabani/HomeWork_GenerasiGIGIH
  
- Go to the project directory
  bash
     cd HomeWork_GenerasiGIGIH
  
- Install dependencies
  bash
     npm install
     npm create-react-app my-app
  
- Set up the environment variables
  `REACT_APP_API_KEY` contains Spotify Client ID
- Start the server
  bash
     npm start
  
- Open `http://localhost:3000/` with your browser to see the result

# Testing

Run `npm test` for launches the test runner in the interactive watch mode.
